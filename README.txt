This is the code for the paper "Modelling Spontaneous Firing Activity of the Motor Cortex in a Spiking Neural Network with Random and Local Connectivity"
Author: Lysea Haggie
Email: lmun373@aucklanduni.ac.nz

To run the simluation, you need to install brian2 - https://brian2.readthedocs.io/en/latest/introduction/install.html

Run the runMain.py to run a simulation and produce figures as shown in the paper (can change connectivity from random to spatial in line 34)
